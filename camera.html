Giraffe PRO Camera

Built for listing capture: one guided property capture collects the data for HDR photos, floor plans, virtual tours, videos, and more.

LiDAR precision and the data behind every output

Built-in LiDAR maps each room as you move—measuring walls, openings, and volume so floor plans, 3D views, and listing geometry stay aligned with what is on site, not guesswork from photos alone.

That depth and spatial data feeds the same capture pipeline as your imagery: one walkthrough collects the measurements and context Content Studio needs for accurate plans, tours, and marketing-ready assets.

One capture. A complete media package

Capture the property once and create the data for HDR photos, floor plans, virtual tours, videos, and other listing-ready outputs.

Every capture goes straight into Content Studio, where you refine, enhance, and publish polished listing assets from one workspace.

Auto-height tripod

The motorized auto-height tripod keeps camera height consistent and enables double-capture mode for hero spaces—richer capture data and stronger outputs where it matters.

Use double-capture mode for standout rooms and advanced outputs; use standard capture everywhere else.

Interactive 3D camera model.
Includes: capture lens / LiDAR / guided capture screen / auto-height tripod connection / cloud upload / battery
